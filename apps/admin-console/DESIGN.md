## Components

### Sidebar Nav Item
- Default: muted text, transparent background
- Hover: `--color-surface2` background, brighter text
- Active: `rgba(34,197,94,0.1)` background, `1px solid rgba(34,197,94,0.15)` border, accent text
- Badge (count): green pill, right-aligned

### Product Row (demo-hub Products page)
```
[⠿ drag] [product_key badge] [Display Name + Description] [↑][↓] [Edit] [Toggle]
```
- Background: `--color-surface`
- Hover: border brightens to `--color-border-hi`
- Disabled row (toggle off): `opacity: 0.5`
- `product_key` badge: monospace code style, muted background — **read-only, not clickable**
- Toggle: green when enabled, gray when disabled, `150ms` transition

### Restart Notice Banner
- Position: TopBar right
- Style: amber dot + muted text — "Changes take effect after restarting demo-hub"
- Non-blocking — informational only

### Edit Modal (for display_name / description)
- Appears on "Edit" button click
- Fields: Display Name (text), Description (textarea)
- Confirms: Save / Cancel
- `product_key` shown as read-only reference
- On save: PATCH `/api/demo-hub/products/:id`

## Authentication
- **Provider**: Supabase Auth (email + password)
- **Flow**: Unauthenticated → redirect to `/login` → Sign in → redirect to `/demo-hub`
- **Session**: Supabase JWT, stored in `localStorage` via Supabase SDK
- **No signup**: Admin accounts created via Supabase dashboard directly
- **Route guard**: React context checks session on every protected route
