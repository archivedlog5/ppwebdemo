## Vault 命名规范说明

| 后缀 | 含义 |
|------|------|
| `-with-purchase` | 首次购买同时完成 Vault 签约 |
| `-setup-only` | 纯 Vault 签约，无购买（setup token，zero-dollar auth） |
| `-return` | 已签约的回头买家使用 Vault 支付的体验 |

## 多账户策略

部分产品需要美国账户（Venmo），其余用中国账户。

### 环境变量

```env
PAYPAL_CN_CLIENT_ID=...        # 中国 sandbox 账户
PAYPAL_CN_CLIENT_SECRET=...
PAYPAL_US_CLIENT_ID=...        # 美国 sandbox 账户（Venmo 专用）
PAYPAL_US_CLIENT_SECRET=...
```

### `buttons` 页面 — 同页多 SDK 实例加载

`/paypal/jssdk-v5/buttons` 是唯一需要同时加载两个 SDK 实例的页面：

```html
<!-- 中国账户 — PayPal / PayLater / BCDC -->
<script
  src="https://www.paypal.com/sdk/js?client-id=CN_ID&components=buttons"
  data-namespace="paypalCN">
</script>

<!-- 美国账户 — Venmo -->
<script
  src="https://www.paypal.com/sdk/js?client-id=US_ID&components=buttons&enable-funding=venmo"
  data-namespace="paypalUS">
</script>
```

```js
// 各自用对应的 namespace 渲染
paypalCN.Buttons({ fundingSource: paypalCN.FUNDING.PAYPAL }).render('#btn-paypal')
paypalCN.Buttons({ fundingSource: paypalCN.FUNDING.PAYLATER }).render('#btn-paylater')
paypalCN.Buttons({ fundingSource: paypalCN.FUNDING.CARD }).render('#btn-bcdc')
paypalUS.Buttons({ fundingSource: paypalUS.FUNDING.VENMO }).render('#btn-venmo')
```

其余所有产品页面只加载中国账户 SDK（单实例）。
