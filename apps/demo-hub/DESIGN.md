## Design System

### Theme
- **Primary**: Dark OLED (`#0F172A` background, `#020617` deep background)
- **Mode**: Dual — Dark (default) + Light mode, toggleable
- **Accent**: `#22C55E` (green) — used for active states, links, status indicators
- **Destructive**: `#EF4444`

### Spacing Scale
`4 / 8 / 12 / 16 / 20 / 24 / 28 / 32 / 48px` — 4pt rhythm

### Effects
- Card hover: `translateY(-2px)` + `box-shadow: 0 8px 24px rgba(0,0,0,0.4)`
- Status dot: `box-shadow: 0 0 6px var(--color-accent)` (glow effect)
- Transition: `150ms ease-out` for all interactive states
- Card top border: 2px gradient strip per provider (brand color)

## Components

### Product Card
- Background: `--color-surface`
- Border: `1px solid --color-border`
- Border-radius: `10px`
- Top accent: `2px gradient` (provider brand color)
- Hover: lift + border brighten
- Content: provider tag → product name → description → "Run demo →" + status dot

### Tab Bar (Demo / Code / Logs)
- Active tab: `--color-accent` bottom border + accent text color
- Disabled tab: `--color-border-hi` text + `cursor: not-allowed`
- Future tabs show "soon" badge in muted style

### Payment Widget Sandbox
- Container: `--color-surface` card, `border-radius: 14px`
- Amount display: Space Mono, 36px, centered
- "Sandbox Mode" badge: green with lightning bolt icon
- Input fields: `--color-bg` background, `--color-border` border
- Pay button: **provider brand color** (per-product decision, deferred)
- Test card hint below button: muted text

### Theme Toggle
- Position: TopBar right area
- Sun/moon icon, smooth `150ms` transition
- Persisted to `localStorage`

## Accessibility Checklist
- [ ] Contrast ratio ≥ 4.5:1 for all text (both light and dark modes)
- [ ] Focus rings visible on all interactive elements
- [ ] Tab order follows visual order
- [ ] `aria-label` on icon-only buttons
- [ ] `prefers-reduced-motion` respected for all transitions
- [ ] Touch targets ≥ 44×44px on mobile
- [ ] No horizontal scroll on any breakpoint
