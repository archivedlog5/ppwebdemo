## Design System

### Theme
- **Primary**: Dark OLED (`#0F172A` background, `#020617` deep background)
- **Mode**: Dual — Dark (default) + Light mode, toggleable
- **Accent**: `#22C55E` (green) — used for active states, links, status indicators
- **Destructive**: `#EF4444`

### Spacing Scale
`4 / 8 / 12 / 16 / 20 / 24 / 28 / 32 / 48px` — 4pt rhythm

### Effects
- Card hover: `translateY(-2px)` + `box-shadow: 0 8px 24px rgba(0,0,0,0.4)`
- Status dot: `box-shadow: 0 0 6px var(--color-accent)` (glow effect)
- Transition: `150ms ease-out` for all interactive states
- Card top border: 2px gradient strip per **sdk_version**（同 provider 品牌色系深浅档；2026-06-10 由「per provider」改为「per sdk」）
- Provider section（首页）: 4px 品牌色左边框 + 极淡品牌色背景带（rgba 品牌色 ~0.05），构成「色块分区」
- SDK label（首页）: chip 形态——同色系淡底 + 该 sdk 深浅色左圆点

### Homepage Structure

三层视觉层级（2026-06-10 重构后，B 色块分区）：
**Provider 大类（22px 亮白 + 品牌色左边框/背景带）→ SDK 小类（13px chip + 同色系深浅）→ 卡片（按 sdk 深浅色调）**。
层级权重：Provider 22px > SDK 13px chip > 卡片标题 13px（尺寸 + 颜色 + 缩进三重信号）。

```
┌─────────────── TopBar (logo + theme toggle) ────────────────────────┐
├─────────────── Hero (title + stats: N demos / M providers) ─────────┤
┃▌PAYPAL ●                                              2 demos    ▸    ┃  ← 4px 品牌色左边框 + 淡背景带
┃▌   ◗ jssdk-v5 ▸          ← sdk chip（同色系淡底 + 深浅色圆点）        ┃
┃▌     [Card][Card][Card][Card]   ← 卡片按 v5 浅蓝色调                  ┃
┃▌   ◗ jssdk-v6 ▸                                                       ┃
┃▌     [Card][Card]               ← 卡片按 v6 深蓝色调                  ┃
┣━ BRAINTREE ● ────────────────────────────────────────────────────────┫
┃▌   ◗ server-sdk ▸  [Card][Card]                                       ┃
├─ STRIPE / ADYEN ("Coming soon" dimmed，同分区样式) ──────────────────┤
└──────────────────────────────────────────────────────────────────────┘
```
- 文字对比度约束：provider/sdk 文字用 `--color-fg` / `--color-fg-muted`，品牌色只作装饰块（边框/背景/圆点），不作文字色 → 兼容浅色主题。

## Accessibility Checklist
- [ ] Contrast ratio ≥ 4.5:1 for all text (both light and dark modes)
- [ ] Focus rings visible on all interactive elements
- [ ] Tab order follows visual order
- [ ] `aria-label` on icon-only buttons
- [ ] `prefers-reduced-motion` respected for all transitions
- [ ] Touch targets ≥ 44×44px on mobile
- [ ] No horizontal scroll on any breakpoint

## Resolved Design Decisions (plan-design-review 2026-05-18)

| 决定 | 结论 |
|------|------|
| 移动端侧边栏 | **水平滚动 tabs** — 可完全自定义样式，与 Dark OLED 设计语言一致 |
| 首页搜索框 | **删除** — 无搜索逻辑支撑，装饰性搜索框损耗用户信任 |
| SDK 加载状态 | **简单 spinner 占位** — 按钮容器内显示 spinner + "Loading..." 文字，SDK ready 后替换 |
| 支付成功/失败消息 | 保持现有 `.result-msg.success` / `.result-msg.error` 样式（已在 sandbox.css 定义）|
